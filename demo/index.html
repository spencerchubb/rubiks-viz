rubiks viz demo

Features and fun facts:

  • Rubik's Cube visualizer using WebGL. Just need one line of javascript to add a cube to any website
  • Can be used in any Javascript framework (React, Angular, Vue, Svelte, etc) because you just need a reference to a div
  • Any size cube (1x1, 2x2, 3x3, 4x4, 5x5, etc)
  • Can render multiple cubes in a page
  • Turn by clicking and dragging
  • Turn with keyboard
